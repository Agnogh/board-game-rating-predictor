# Board Game Rating Predictor

## Project Overview

Board Game Rating Predictor is a predictive analytics project focused on exploring and predicting board game ratings using machine learning.

The project will use board game data to investigate how features such as player count, play time, age rating, complexity, categories, and mechanics may relate to user ratings and popularity.

## Project Goals

- Collect and prepare a board game dataset
- Explore patterns in board game ratings and popularity
- Clean and transform the data for machine learning
- Build a predictive machine learning model
- Evaluate model performance
- Create a Streamlit dashboard for presenting insights and predictions

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit
- Jupyter / VS Code notebooks
